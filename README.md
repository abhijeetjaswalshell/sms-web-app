# SMS web app
_Using React & Firebase_

## Installation

### 1. Clone repository and step into directory
```
git clone git@github.com:danolife/sms-web-app.git
cd sms-web-app
```

### 2. Install dependencies
```
npm install
```
or
```
yarn install
```

### 3. Create production build
```
npm run build
```
or
```
yarn build
```
